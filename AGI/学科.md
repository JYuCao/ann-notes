# 1

## 1️⃣ World Models / Predictive Coding

**核心概念：**

* 模型不仅学“输入→输出”，而是学 **世界本身的动态规律** 。
* AGI 需要能“想象未来、预测结果”，而不是只是模仿过去。

**关键点：**

* **World Model** ：构建一个内部环境模型，模拟因果和状态转移。
* **Predictive Coding** ：通过不断预测感知输入，然后用预测误差调整模型内部表征。

**代表方法 / 例子：**

* Ha & Schmidhuber 的  *World Models* （2018）: 用 VAE + RNN 构建环境模拟器，让 RL agent 在模拟世界中训练。
* Rao & Ballard 的  *Predictive Coding Theory* （1999）：大脑皮层可能就是通过预测输入信号的误差来更新内部表征。

**AGI 价值：**

* 允许智能体**内部模拟**未来，进行规划和假设检验。
* 是**理解环境、泛化能力和少样本学习**的核心工具。

## 2️⃣ Causal Representation Learning（因果表示学习）

**核心概念：**

* AGI 不仅要学统计相关性（correlation），还要学 **因果关系（causation）** 。
* 通过学习因果结构，智能体能回答**“如果我做 X，会发生什么？”**的问题，而不是只预测历史模式。

**关键点：**

* 从观测数据中发现变量之间的因果关系。
* 将原始高维感知映射到 **因果变量空间** （disentangled + causal latent space）。

**代表方法 / 例子：**

* Pearl 的因果图（Causal Graphs）
* DeepMind / MIT 的 **Causal Representation Learning** 项目

  → 学习可干预、可预测的 latent factor

**AGI 价值：**

* 支撑 **通用规划与决策** ，不仅能预测结果，还能干预世界。
* 解决**分布转移和反事实推理**问题（也就是你在 adversarial / rationale 里看到的核心问题）。

## 3️⃣ Agent-based Learning（RL + Memory + Planning）

**核心概念：**

* AGI 是 **主动的智能体** ，而不是被动的函数映射。
* 核心能力：
  * RL：学行动策略（policy）
  * Memory：记住历史状态与因果链
  * Planning：在内部模拟中做长远决策

**关键点：**

* 结合强化学习 + 内部记忆（LSTM, Transformer memory）
* 对复杂环境的长期规划能力
* 能在部分可观测环境中推理（POMDP）

**代表方法 / 例子：**

* DeepMind 的  *Neural Map* , *Memory-Augmented RL*
* AlphaZero / MuZero：内部环境建模 + 长期规划
* RL² / Meta-RL：学会“快速适应新任务”

**AGI 价值：**

* 让智能体 **具备自主探索、计划和长期目标导向行为** 。
* 是“动作 + 环境交互”的基础能力。

---

## 4️⃣ Symbol–Neural Hybrid（符号–神经混合）

**核心概念：**

* 结合**符号 AI 的逻辑、推理能力**与 **神经网络的感知、模式识别能力** 。
* 不等于老派 symbolic AI，核心是 **融合而非替代** 。

**关键点：**

* 神经网络用于处理复杂感知（图像、语音、文本）。
* 符号系统用于处理 **抽象逻辑、规则推理和规划** 。
* 神经部分和符号部分可以相互反馈（neuro-symbolic reasoning）。

**代表方法 / 例子：**

* DeepMind 的 *Neuro-Symbolic Concept Learner (NS-CL)*
* IBM / MIT 的 *Differentiable Neural Computers + Logic Modules*
* OpenAI 的部分 LLM 结合 external symbolic reasoning 的实验

**AGI 价值：**

* 支撑 **抽象思维、因果推理和可解释决策** 。
* 是解决**少样本学习、反事实推理和可解释性**的关键方向。
