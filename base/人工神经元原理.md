# 人工神经元原理

## MP神经元
- McCulloch-Pitts Neuron，数学家
- $ y = \text{sgn}( \sum_{i=1}^{n} (w_i x_i - \theta) ) = \text{sgn}( w_i x_i + b ) $
- 本质是一个 线性分类模型，作用是对输入进行二分类。
- 权重调整：根据经验和知识来手动调整这些权重。

## 单层神经网络（感知器）
- 感知器 Perceptron：类似一个逻辑回归模型，用于做线性分类任务


- 线性回归模型
    - $ y = \sum_{i=1}^{n} w_i x_i + b $
    - 主要目标是找到最佳参数预测 y hat 与 y 之间的最小误差（MSE）。
    - 回归：找到一个最佳拟合线来预测线性连续值。

- 逻辑回归模型
    - $ P(y=1) = \frac{1}{1 + e^{-\left( \sum_{i=1}^{n} w_i x_i + b \right)}} $
    - 用于解决二分类问题，引入了 Sigmoid 将输出映射到 (0, 1) 之间。
    - 逻辑回归的主要目标是找到最佳的参数，使得概率 P(y=1) 最大化（最大似然估计）。

- 自动调整参数
    - 有监督学习：模型通过学习已知标签的训练数据集，来预测新数据的标签。

    - 无监督学习：无监督学习的训练数据集中不包含标签信息，因此模型需要自行发现数据的内在结构和模式。

    - 误差修正方法：用于减少在机器学习中，由于各种因素（如数据噪声、模型复杂度等）的影响，模型的预测结果可能会存在的误差。

## 两层神经网络（多层感知器）
- 多层感知器（MLP）：引入了隐藏层，能够学习更复杂的非线性关系。
- 非线性激活函数
    - $ sigmoid(x) = \frac{1}{1 + e^{-x}}$
    - $ tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}} $
    - $ ReLU(x) = max(0, x) $
    - $ Leaky ReLU(x) = max(0.01x, x) $
- 万能逼近定理
    - 一个具有足够隐藏层和神经元的两层神经网络可以以任意精度逼近任何连续函数。
    - 线性变换进行升维/降维、放大/缩小、旋转和平移（偏置）；非线性变换进行弯曲。
    - 隐藏层对输入空间进行非线性变换，输出层负责生成线性决策边界。
- 隐藏层节点数
    - 隐藏层节点数的选择是一个重要的超参数，过多可能导致过拟合，过少可能导致欠拟合。
    - 网格搜索（Grid Search）：预先设定几个可选值，通过切换这几个值来看整个模型的预测效果，选择效果最好的值作为最终选择。
- 梯度下降+BP算法
    - 反向传播算法（Backpropagation）：通过链式法则计算损失函数相对于每个权重的梯度，并使用这些梯度来更新权重。
    - 梯度下降：通过迭代更新权重来最小化损失函数，常用的优化算法包括随机梯度下降（SGD）、Adam等。

## 多层神经网络（深度学习）
- 预训练与微调
    - 预训练（Pre-training）：在大规模数据集上训练一个深层神经网络，以学习通用特征，然后将这些特征迁移到其他相关任务中。
    - 微调（fine-tuning）：在预训练模型的基础上，使用较小的数据集进行进一步训练，以适应特定任务。

- 为什么越深越好？
    - 更深入的表示特征：深层神经网络能够学习更抽象和复杂的特征表示，从而提高模型的表达能力。
    - 更强的函数模拟能力：更多的参数意味着其模拟的函数可以更加的复杂，可以有更多的容量（capcity）去拟合真正的关系。

- 基本工作原理：前向传播 -> 计算损失 -> 反向传播 -> 更新权重

- 梯度下降与反向传播

- 训练技术（主题：优化、泛化）
    - 优化：新的算法（如一种带动量因子的梯度下降算法）、更好的权重初始化方法（如Xavier初始化、He初始化）等
    - 泛化：正则化、dropout、数据扩容（data-augmentation）等
